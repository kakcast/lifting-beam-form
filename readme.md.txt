# Lifting Beam Inspection System

A web-based system for managing lifting beam inspections, with user authentication, form submission, PDF generation, and report management.

## Features

- **User Authentication**: Secure login system with role-based access control
- **Digital Inspection Forms**: Complete lifting beam inspection forms with digital signatures
- **PDF Generation**: Download completed forms as PDF documents
- **Report Management**: View, filter, and manage inspection reports
- **User Management**: Admin panel for managing system users

## Files

- `index.html` - Main inspection form
- `login.html` - User login page
- `admin.html` - User management for administrators
- `reports.html` - View and manage inspection reports
- `style.css` - Styling for all pages
- `script.js` - JavaScript functionality
- `test.html` - Simple test page to verify GitHub Pages setup

## Setup

1. Clone this repository
2. Enable GitHub Pages in repository settings
3. Access the system through the provided GitHub Pages URL

## Default Credentials

- Username: admin
- Password: admin123

## Usage

1. Log in using the credentials above
2. Fill out inspection forms
3. Generate PDF reports
4. Manage users (admin only)
5. View and filter inspection reports

## Notes

- This system uses client-side storage (localStorage) for data persistence
- In a production environment, a proper backend server and database should be used
- All user data and reports are stored in the browser and will be lost if browser storage is cleared
